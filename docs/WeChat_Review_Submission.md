## 2. 小程序基本信息

### 2.1 基本信息
- **小程序名称**：戒色助手
- **英文名称**：NoFap Helper
- **简介**：专注于帮助用户建立健康生活习惯的助手工具
- **服务类目**：工具 > 健康管理
- **标签**：健康管理、习惯养成、自我提升

### 2.2 详细描述
戒色助手是一款专业的健康生活习惯管理工具，通过科学的方法帮助用户建立健康的生活方式。主要功能包括：

- **习惯追踪**：每日打卡记录，进度可视化展示
- **专业评估**：科学的自我评估系统和分析报告  
- **学习成长**：心理健康、行为改变等专业知识内容
- **社区支持**：用户交流互助，经验分享平台
- **紧急求助**：危机干预机制，专业心理支持
- **激励系统**：游戏化激励，成就解锁机制

### 2.3 目标用户
- 希望改善生活习惯的成年用户
- 关注心理健康和自我提升的人群
- 需要社区支持和专业指导的用户

## 4. 技术架构

### 4.1 前端技术
- **框架**：uni-app X（支持多端发布）
- **UI库**：自研组件库 + uni-ui
- **样式**：Tailwind CSS
- **状态管理**：Pinia
- **网络请求**：uni.request

### 4.2 后端技术  
- **框架**：Gin（Go语言）
- **数据库**：MySQL 8.0
- **缓存**：Redis
- **认证**：JWT + 微信登录
- **部署**：Docker + Kubernetes

### 4.3 安全措施
- **数据加密**：AES-256-GCM
- **传输安全**：HTTPS + TLS 1.2+
- **访问控制**：RBAC权限管理
- **防护机制**：XSS、SQL注入、CSRF防护
- **隐私保护**：数据脱敏、匿名化处理
