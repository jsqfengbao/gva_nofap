# 戒色助手微信小程序开发项目 PRD

## 项目概述
本项目旨在开发一个专注于帮助年轻人戒除色情内容依赖的游戏化健康管理微信小程序。项目包含三个主要部分：

1. **前端小程序**：使用uni-app X (Vue 3 + TypeScript)开发
2. **后端服务**：基于现有Gin框架的Go服务，扩展支持小程序功能
3. **管理端Web**：基于现有Vue框架，改写完成管理功能

## 技术架构
- **前端技术栈**：uni-app X、Vue 3、TypeScript、Tailwind CSS、Pinia
- **后端技术栈**：Go 1.22+、Gin、MySQL 8.0+、Redis、GORM、JWT
- **管理端技术栈**：Vue 3、Element Plus、TypeScript

## 核心功能模块

### 1. 用户管理系统
- 微信授权登录
- 用户信息管理
- 隐私设置
- 数据同步

### 2. 色隐指数评估系统
- 50题科学评估问卷
- 5级风险等级判定(正常、轻度、中度、重度、严重)
- 评估历史记录
- 定期复评提醒

### 3. 游戏化激励系统
- 50级等级系统
- 经验值奖励机制
- 100+成就徽章系统
- 连续打卡奖励
- 里程碑奖励

### 4. 每日打卡功能
- 每日签到打卡
- 情绪状态记录(1-5级)
- 连续天数统计
- 打卡历史查看
- 打卡奖励机制

### 5. 社区互助功能
- 匿名发帖系统
- 内容分类(经验分享、求助求鼓励、日常打卡、成功故事)
- 点赞评论功能
- 内容审核机制
- 用户互动统计

### 6. 紧急求助系统
- 一键紧急求助
- 在线志愿者响应
- 匿名求助发布
- 实时互助聊天
- 危机干预记录

### 7. 学习内容模块
- 个性化推荐系统
- 文章模块(科普、康复指导、心理健康)
- 视频模块(专家讲座、经验分享)
- 音频模块(冥想指导、正念练习)
- 学习进度追踪
- 内容收藏功能

### 8. 数据分析与报告
- 进度追踪和可视化
- 连续天数、成功率统计
- 情绪变化趋势分析
- 个人成长报告生成

### 9. 管理端功能
- 用户管理和数据统计
- 内容管理(文章、视频、音频)
- 社区内容审核
- 评估问卷管理
- 成就系统管理
- 数据分析Dashboard

## 数据库设计
基于MySQL 8.0+设计，包含以下核心表：
- users(用户表)
- abstinence_records(戒色记录表)
- daily_checkins(每日打卡表)
- assessment_results(评估结果表)
- community_posts(社区动态表)
- community_comments(社区评论表)
- emergency_helps(紧急求助表)
- help_responses(求助响应表)
- learning_contents(学习内容表)
- user_learning_records(用户学习记录表)
- achievements(成就系统表)
- user_achievements(用户成就表)

## 开发阶段规划

### Phase 1: 基础架构搭建(2-3周)
- 前端uni-app X项目初始化
- 后端API扩展和数据库设计
- 管理端基础框架搭建
- 用户认证系统实现

### Phase 2: 核心功能开发(4-6周)  
- 色隐指数评估系统
- 每日打卡功能
- 游戏化激励机制
- 基础UI组件库

### Phase 3: 高级功能开发(4-6周)
- 社区互助功能
- 紧急求助系统
- 学习内容模块
- 数据分析系统

### Phase 4: 管理端和优化(2-4周)
- 管理端功能完善
- 性能优化和安全加固
- 全面测试和部署
- 小程序审核准备

## 质量要求
- 响应时间：页面切换 ≤ 1秒，数据加载 ≤ 2秒
- 兼容性：支持iOS 12+、Android 6.0+
- 安全性：数据加密、用户隐私保护
- 可用性：用户满意度 ≥ 4.5/5.0
- 性能：应用安装包 ≤ 150MB，运行内存 ≤ 200MB

## 商业目标
- 6个月内DAU达到1万+
- 30天戒色成功率 > 60%
- 用户7日留存率 > 45%
- 社区日发帖量 > 500条

## 风险控制
- 严格的内容审核机制
- 用户隐私保护措施
- 微信小程序合规性审查
- 技术架构的可扩展性设计

此项目将为用户提供专业、安全、有效的戒色辅助工具，通过科学的评估体系和游戏化机制帮助用户建立自控力，摆脱不良习惯，重建健康的生活方式。
